# 介绍

My Uni 是专为 uni-app 开发的摸鱼插件库，为你搜集了丰富的优秀 uni-app 插件，助你轻松工作，天天摸鱼！同时我们也开发并提供了包含路由、CI/CD、wot-ui代码提示等插件，基于 uni-app 能力，对标 web 流行技术栈，提供近似的开发体验，支持 APP、H5、微信小程序 等多平台。

提供包含路由、CI/CD、wot-ui代码提示等插件并搜集了丰富的优秀 uni-app 插件，助你轻松工作，天天摸鱼！

## 主要特点

- 💪 **支持 TypeScript** - 使用 Typescript 构建，提供良好的组件类型系统
- 🆒 **改善 uni-app 开发体验** - 基于 uni-app 能力，对标 web 流行技术栈，提供近似的开发体验
- 🚦 **Router** - 基于 uni-app 路由能力，提供近似的路由体验
- 🔧 **CI/CD** - 基于 uni-app 能力，提供简单的 CI/CD 功能
